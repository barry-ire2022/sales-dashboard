# Sales Dashboard

An **interactive sales dashboard** built with Python and Dash.  
Visualize your sales data with filters for Region, Category, and Salesperson, and get key metrics like Total Sales and Average Sale.

## Features

- Filter sales data by **Region**, **Category**, or **Salesperson**
- View **sales trends over time**
- Analyze **sales by category**
- Dashboard runs locally in a web browser
- Click **Exit Dashboard** to stop the server

## Getting Started

### Prerequisites

- Python 3.10+
- pip

### Install Dependencies

Create a `requirements.txt` with the following:
pandas==2.1.0
dash==2.13.0
plotly==5.18.0
werkzeug==2.3.10
